# Anthony Campolo, Developer Advocate
Hello 👋, my name is Anthony Campolo. I'm an open source Developer Advocate, former member of the [RedwoodJS](https://rwsdk.com/) core team, and co-host of the [FSJam Podcast](https://fsjam.org/). I'm currently working as a Collaborative Engineer at [Atmosera](https://www.atmosera.com/) where I teach enterprise devs how to use AI coding tools.
This website is built with a Bun-powered static generator. My content mostly focuses on web development frameworks and deployment platforms. Other topics I cover include AI and large language models. This is my blog, continue reading to achieve peak blogness.
